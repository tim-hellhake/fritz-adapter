Connect your Fritz! devices.

## Configuration
Add the `username` and the `password` of a user with Smart Home access.
The default for `host` is `http://fritz.box`.

## Supported devices
- FRITZ!DECT 200
- FRITZ!DECT 301

## Teach in device
- Go to the _Add things_
- A new thing should appear
- Add it
